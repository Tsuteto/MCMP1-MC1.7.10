# MCMP-1
レコーダーで好きな音源をカセットにダビングしてプレーヤーで聴くスタイルのポータブル音楽プレーヤー。だだっ広いマインクラフトの世界、音楽を持ち歩こう。

## Distribution and Usage
https://tsuteto.hacca.jp/mcmods
